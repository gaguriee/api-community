# Spring Boot - Basic Community Service REST API 구현

### 개요
- 기본적인 커뮤니티 기능을 적용한 REST API 서비스 구현

### 개발환경
- IntelliJ
- Spring Boot 3.1.0
- JDK 17
- JPA & MySQL
- Gradle 7.6.1

### 개발목표
- JWT 토큰 + Spring Security를 이용한 사용자 인증 및 인가
- Redis (Refresh Token 저장용)
- OAuth (Kakao, Google)
- AWS (EC2, S3) / CDN을 이용한 이미지,동영상 업로드
